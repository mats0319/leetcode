# leetcode
一个心比天高的项目：刷完leetcode的1000多道题。相关知识点总结，会更新在[简书](https://www.jianshu.com/u/ef1998502270)。
### 定位
作为一个吃这碗饭的人，刷leetcode的题是为了熟悉go语言、积累使用经验，以及编程技巧和简单算法**学习**。  
而以学习为目的，可以预见的，进度不会很快，甚至会因为重构代码而“开进度的倒车”。  
### 结构
拟划分结构为题目解答与扩展学习；  
- 题目解答：每一道题对应一个文件夹；
- 扩展学习：记录一些题目衍生的内容。
